1. Explain why we need long long int data type? 
Ans : long long int is used to store very large integer values that exceed the range of int or long int.
Example: int may store up to 4 byte , but long long int can store up to 8 byte quintillion.

3. Write a C program that will take 2 numbers from the user and then print the 2nd number first and then first number.

    int a, b;
    printf("Enter two numbers: ");
